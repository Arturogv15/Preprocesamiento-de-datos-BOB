# Plan Summary - Quick Reference

## What We're Building
Interactive web platform for teaching data preprocessing in data mining courses.

## Tech Stack
- **Backend**: Flask + Python
- **Frontend**: HTML/CSS/JavaScript + Bootstrap 5
- **Data**: pandas, scikit-learn, matplotlib, plotly
- **Database**: SQLite (dev) / PostgreSQL (prod)

## Core Features (MVP)
1. ✅ User authentication (register/login)
2. ✅ Dataset upload & preview
3. ✅ Data cleaning (missing values, duplicates, outliers)
4. ✅ Data transformation (scaling, encoding)
5. ✅ Data integration (joins, merging)
6. ✅ Dimensionality reduction (PCA, feature selection)
7. ✅ EDA visualizations (plots, heatmaps, statistics)
8. ✅ Theory content (4 modules)
9. ✅ Quiz system
10. ✅ Progress tracking

## Project Structure
```
Preprocesamiento/
├── app/                    # Flask application
│   ├── __init__.py
│   ├── models.py          # Database models
│   ├── routes.py          # Main routes
│   ├── auth.py            # Authentication
│   ├── data_processing.py # Data operations
│   └── visualization.py   # Plots generation
├── static/                # CSS, JS, datasets
├── templates/             # HTML templates
├── config.py
├── requirements.txt
└── run.py
```

## Implementation Priority
1. **Week 1**: Setup + Authentication + Basic UI
2. **Week 2**: Dataset management + EDA tools
3. **Week 3**: Data cleaning module
4. **Week 4**: Transformation + Integration
5. **Week 5**: Dimensionality reduction + Theory
6. **Week 6**: Quiz system + Progress tracking
7. **Week 7**: Polish + Advanced features
8. **Week 8**: Testing + Deployment
